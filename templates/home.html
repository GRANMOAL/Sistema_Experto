Sistema Experto Hardware

Sistema para entrenar y usar modelos de Machine Learning para predicción de rendimiento de hardware

GUI 1: Entrenar Modelos

Sube un archivo CSV, selecciona el modelo (ID3 o KNN), configura los parámetros de validación y entrena tu modelo.

Funcionalidades:
  • Carga de archivos CSV
  • Selección de modelo (ID3/KNN)
  • Configuración de Hold-out o K-Folds
  • Entrenamiento y evaluación
Entrenar Modelo
GUI 2: Realizar Predicciones

Introduce las características del hardware y obtén predicciones usando los modelos entrenados.

Características requeridas:
  • MYCT - Tiempo de ciclo de máquina
  • MMAX - Memoria principal máxima
  • CACH - Memoria caché
  • CHMAX - Canales máximos
  • ERP - Rendimiento relativo estimado
Hacer Predicción
