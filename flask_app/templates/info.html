Cómo usar el Sistema Experto
GUI 1: Entrenamiento
  1. Prepara un archivo CSV con tus datos
  2. Sube el archivo usando el formulario
  3. Selecciona el modelo (ID3 o KNN)
  4. Configura el método de validación
  5. Ajusta los parámetros según sea necesario
  6. Haz clic en "Entrenar Modelo"
  7. Revisa los resultados del entrenamiento
GUI 2: Predicción
  1. Selecciona un modelo entrenado
  2. Ingresa los valores para cada característica
  3. Verifica que todos los campos estén completos
  4. Haz clic en "Realizar Predicción"
  5. Obtén el resultado de la predicción
Requisitos del archivo CSV:
  • Formato CSV válido con separadores de coma
  • Primera fila con nombres de columnas
  • La última columna debe ser la variable objetivo
  • Sin valores faltantes o vacíos
  • Datos numéricos para todas las características
